
Vmaj Dreamcast sound Emulator v1.03


Enjoy playing Dreamcast ADX files with WinAmp

How to???

Load a .ADX file from a Dreamcast CD.
If you can't find any Adx files try to find some *.Afs file.

Installation

Copy In_Adx.dll into your WinAmp plugins directory
